# Contributing

Thank you for helping improve Euroleague-Tech-data.

## Contribution Scope

Contributions are welcome for:
- correcting factual inaccuracies
- adding missing public sources
- improving structure and consistency
- expanding team/vendor profiles with verifiable evidence

## Rules for Data Contributions

1. Keep facts and assumptions clearly separated.
2. Add source links when introducing new claims.
3. Avoid marketing language and unverifiable statements.
4. Do not upload copyrighted documents unless redistribution rights are clear.
5. Keep file naming conventions consistent with existing folders.

## Pull Request Guidelines

1. Create a branch from `main`.
2. Keep PRs focused and small.
3. Explain what changed and why.
4. Include source references in the PR description when applicable.

## Code of Conduct

By participating, you agree to follow `CODE_OF_CONDUCT.md`.
