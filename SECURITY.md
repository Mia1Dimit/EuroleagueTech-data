# Security Policy

## Reporting a Vulnerability

Please do not disclose security issues publicly in issues.

Report privately to maintainers with:
- summary of the issue
- impact
- reproduction steps
- suggested mitigation (optional)

## Scope

This repository is primarily data/documentation. Security scope includes:
- accidental exposure of sensitive or private information
- malicious data payloads in contributed files
- unsafe links or embedded content

## Data Privacy

If you identify potentially sensitive personal data, report it immediately and avoid redistributing it.
